# 🔷 Git Workflow

## 📌 What is it?
Process of managing code changes using branches

## 🎯 Why we use it?
To avoid breaking main code and work safely

## ⚙️ Steps
1. Create feature branch
2. Commit changes
3. Push to GitHub
4. Create Pull Request
5. Merge to main

## 💻 Commands
git checkout -b feature/login
git add .
git commit -m "Add login"
git push origin feature/login

## ⚠️ Mistakes
- Working directly on main
- Bad commit messages

## 🧠 My Understanding
Git workflow helps manage features safely

Codify project
